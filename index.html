Everything a SOC Team Needs

From log collection to automated response — CyberNest covers the full security operations lifecycle.

🛡

SIEM — Detect

Real-time log ingestion from agents, syslog, and APIs. 11 format parsers (Syslog, CEF, LEEF, Windows XML, Suricata, CloudTrail, Palo Alto, FortiGate, Cisco ASA). Normalized to ECS schema with GeoIP enrichment.

Correlation Engine

Sigma rule format support + custom YAML rules. Sliding window correlation for brute force, port scans, lateral movement, data exfiltration, DNS tunneling, and C2 beaconing detection.

SigmaSliding WindowMITRE ATT&CK

SOAR — Respond

YAML playbooks with real integrations: VirusTotal, AbuseIPDB, Shodan lookups. Firewall IP blocking, Active Directory user disable, host isolation, Jira tickets, Slack/Teams/PagerDuty notifications.

VirusTotalJiraSlackPagerDuty
🔍

Log Search (Splunk-like)

Full-text search over billions of events with field filters, time ranges, boolean operators, and wildcards. Expandable JSON rows, CSV export, saved searches. Powered by Elasticsearch.

Lucene QueryElasticsearch
📄

Case Management

Full incident lifecycle: create cases from alerts, assign analysts, add tasks and observables (IOCs), track timeline, MITRE ATT&CK tagging. Templates for Ransomware IR, Phishing IR, and more.

TimelineObservablesTasks

Cross-Platform Agent

Lightweight Python agent for Windows, Linux, macOS. Collects Event Logs, syslog, and custom files. File integrity monitoring, process monitoring, network connection tracking. Auto-enrollment with buffering.

Threat Intelligence

Built-in IOC database with STIX/TAXII feed ingestion. Auto-lookup during log parsing. Supports AlienVault OTX, Abuse.ch, FeodoTracker, EmergingThreats, MISP. Custom IOC upload.

Get Started

Deploy in 3 Commands

From zero to a running SIEM + SOAR in minutes with Docker Compose.

# Clone the repository
$ git clone https://github.com/abhiiibabariya-dev/CyberNest.git
$ cd CyberNest

# Configure environment
$ cp .env.example .env

# Launch all 12 services
$ docker compose up -d

# Dashboard: https://localhost
# API Docs: https://localhost/api/docs
# Default: admin / admin123
